# JS-Fundamentals

## Description
This repository contains beginner-level JavaScript exercises.  
The first task demonstrates how to create a constant variable and print it using `console.log()`.

## Task 0: First constant, first print
**Objective:** Write a script that prints “JavaScript is amazing”.

**Requirements:**
- Create a constant variable called `myVar` with the value `"JavaScript is amazing"`.
- Use `console.log(...)` to print the output.
- Do **not** use `var`.

**Example Output:**
```bash
guillaume@ubuntu:~/0x12$ node 0-javascript_is_amazing.js
JavaScript is amazing
